1. Освоение Git и командной разработки
В рамках первой части работы были отработаны следующие навыки:
Основы: инициализация репозитория, работа с commit и подготовка файлов.

Ветвление: создание веток, переключение между ними (master и branch_1) и создание меток (tags).

Слияние и конфликты: демонстрация процесса merge с ручным разрешением возникших конфликтов.

История: анализ дерева изменений и откат правок.

Удаленная работа: синхронизация локального репозитория с GitHub (push/pull).

2. Интерфейсное тестирование OpenBLAS
Вторая часть посвящена созданию тестов для библиотеки линейной алгебры OpenBLAS:
Объект тестирования: функции интерфейса CBLAS (на языке C).
Охват тестов:

Тестирование в различных операционных системах.

Проверка работы в однопоточном и многопоточном режимах.

Валидация всех типов аргументов и возвращаемых значений.

Анализ покрытия: демонстрация полноты покрытия функционала тестами.

Negative Testing: создание «сломанной» версии библиотеки для демонстрации ожидаемых падений тестов.
